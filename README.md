### 🧮 Calculator App

A simple and responsive calculator application developed using **HTML, CSS, and JavaScript**.  
This project was created to practice front-end development fundamentals, DOM manipulation, event handling, and basic user interface design.

**Technologies used:**

- HTML
- CSS
- JavaScript
- GitHub Pages

**Main features:**

- Basic arithmetic operations
- Clear and delete functions
- Responsive calculator layout
- Modern dark-themed interface
- Beginner-friendly JavaScript logic
